# DSA Progress Tracker

A simple Python GUI application built using Tkinter to track DSA problems and progress.

---

## Features
✅ Add Problems  
✅ Delete Problems  
✅ Mark Problems as Solved  
✅ Save Progress Automatically  
✅ Load Saved Progress on Restart  

---

## Tech Stack
- Python
- Tkinter
- File Handling

---

## Project Files
- gui.py → Main GUI application
- progress.txt → Stores saved progress
- README.md → Project documentation

---

## How to Run

1. Open terminal
2. Run:

```bash
python gui.py
